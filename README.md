# Calculator – Pure HTML/CSS/JS

![GitHub license](https://img.shields.io/github/license/kalana-03/Calculator_JavaScript?style=flat-square)
![GitHub repo size](https://img.shields.io/github/repo-size/kalana-03/Calculator_JavaScript?style=flat-square)
![Demo](https://img.shields.io/badge/live-demo-brightgreen?style=flat-square)

A **lightweight, responsive calculator** built with vanilla HTML, CSS, and JavaScript – no frameworks, no dependencies. Handles basic arithmetic with a clean, neumorphic design.

![Calculator screenshot](screenshots/demo.png)

---

## Features

| Checkmark | Feature |
|-----------|---------|
| **Basic arithmetic** | `+  –  ×  ÷` |
| **Decimal support** | `.` |
| **Clear / Backspace** | `AC` & `⌫` |
| **Sign toggle** | `+/-` (toggles negative/positive) |
| **Responsive UI** | Mobile-friendly (<480px auto-adjusts) |
| **Wide zero button** | Full-width `0` for easy use |
| **Neumorphic styling** | Shadows, hovers, and scales for tactile feel |
